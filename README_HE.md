# Lior Trading MCP v1

גרסה ראשונה נקייה של הסורק:
- בלי Google Cloud
- בלי Telegram
- בלי לולאה אינסופית
- מיועד להפעלה דרך ChatGPT MCP בהמשך

Tools:
- `scan_market` — סורק Watchlist ומחזיר setups
- `analyze_stock` — מנתח טיקר יחיד

הקובץ עדיין דורש אירוח MCP HTTPS כדי להתחבר למסך Create app ב-ChatGPT.
לבדיקה מקומית אפשר להריץ שרת FastMCP, ואז לחשוף אותו עם ngrok/Cloudflare Tunnel.
